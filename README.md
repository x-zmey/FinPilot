# LedgerAI - AI-Powered Bookkeeping Platform

An intelligent financial services platform for bookkeeping, bank reconciliation, transaction categorization, and financial reporting.

## Demo Credentials

- **Email:** `demo@ledgerai.com`
- **Password:** `demo1234`

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Features

- AI-powered transaction categorization with confidence scores
- Bank reconciliation workflow with matching interface
- Financial dashboards with KPIs and charts
- Document upload and processing
- Profit & Loss, Balance Sheet, and Cash Flow reports
- Cash flow forecasting with scenario analysis
- Settings with integrations management

## Tech Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- shadcn/ui components (Radix UI)
- Recharts for data visualization
- Framer Motion for animations
- Zustand for state management
- React Router v6 for routing

## Deployment

Configured for Vercel with SPA rewrites via `vercel.json`.
